# BlockchainMiniProject Lacasa De Villa

What is this?

A smart contract that use for rent a villa using ethereum environment.

Actor :
- Landlord
- Tenant

What can Landlord do?
- Build a Villa.
- Demolish the Villa.

What can Tenant do?
- Check if the Villa is Ready for Rent.
- Check Villa status.
- Check In.
- Check Out.
